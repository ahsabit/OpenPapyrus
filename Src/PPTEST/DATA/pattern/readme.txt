Этот каталог предназначен для файлов образцов для сравнения с файлами в рабочих каталогах исходных кодов.
Цель - сравнение рабочих копий файлов с образцами для выявления отсутствия значительных изменений.
Мотивацией для этого послужило неприятное событие - файл ppifc.uuid после передачи данных из одного из
репозиториев был сильно модифицирован, что привело к неработоспособности COM-интерфейсов релиза.
